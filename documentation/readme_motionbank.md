# How to process video using Piecemaker

This is the process to import the performance video into motionbank:

1. Upload the Video (unlisted) to youtube.studio

2. Copy the youtube video link

3. Log into motionbank → Piecemaker

4. Create a timeline for the performance (e.g. GestureLab_Leo)

5. Click on `+ add media` to include the youtube video link.

6. (optional) set actual date/time

Here is how to annotate it:

1. Go to Piecemaker

2. Choose appropriate *timeline*

3. Edit/Annotate

Here is how to export the video/annotation data as a **.csv** file.

1. Go to motionbank → Piecemaker

2. Select the `edit` option for the desired timeline.

3. There, choose, `export` as **.csv** file.
